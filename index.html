Uso correcto fuentes:
  • Utilizar la fuente Salsa solo para banner y títulos
  • Utilizar la fuente Varela solo para botones, subtítulos y títulos
  • Alineación correcta
Uso incorrecto fuentes
  • Utilizar la fuente Salsa en descripciones, botones, mensajes de error, formularios
  • Utilizar la fuente Varela para títulos o banners
  • Mala alineación
Uso correcto colores:
  • Colores que transmitan calma y seriedad
  • Colores coherentes con el conjunto de la marca
  • Utilizar el color principal solo para banner y footer.
  • Utilizar el color secundario solo para fondo.
  • Color de acción en botones para visualizar proyectos
Uso incorrecto colores
  • Colores agresivos
  • Uso del color de acción en banners, footers, formularios o textos
  • Colores que ignoren la accesibilidad
  • Inconsistencia de uso en todo el diseño
